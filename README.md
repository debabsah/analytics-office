# bi-copilot

An expert BI guide for technical practitioners working in the Claude Code TUI — a bench of senior mentors, grown one skill at a time.

## v1: `groundwork` — the project-orientation guide
You inherit an undocumented SSIS/stored-proc/SQL-Agent/DB2i estate (or get a vague ticket, or nothing). `groundwork` reads what it can (code/text only — never your live data), interviews you for the rest, **surfaces what you don't know yet**, and builds a **living knowledge base** — state + an append-only continuity log — that you and AI agents can resume from.

**Before:** a blank page and a pile of someone else's objects.
**After:** a `knowledge-base/` that says what the estate does, what's still unknown, what was decided and why, and "catch me up" on demand.

### Install
Clone into your Claude Code skills/plugins location, or add this repo as a plugin marketplace.
